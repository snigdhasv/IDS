For Most Users:
  → Start with PowerShell (Method 1) - it's already installed!
  → If you need more power later, add Scapy (Method 2)
  → Skip tcpreplay - Scapy can replay PCAPs too!

Quick Start:
  1. Open PowerShell as Admin
  2. Configure network (see WINDOWS_QUICK_SETUP.md)
  3. Run: 1..100 | ForEach-Object { curl "http://192.168.100.1" }
  4. Watch your IDS detect the traffic!
